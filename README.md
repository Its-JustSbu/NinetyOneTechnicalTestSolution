# NinetyOneTechnicalTestSolution

## Setup & Installation
1. Clone the repository:
   `https://github.com/Its-JustSbu/NinetyOneTechnicalTestSolution.git`
2. Install dependencies:
   `dotnet restore`
3. Run the application:
   `dotnet run`
4. Access the API documentation at: `https://localhost:7274/swagger`

## Design Decisions
1. **Architecture:** I used a repository pattern to keep the application logic separate from the API controllers.
2. **Database:** SQLite was chosen for ease of portability and to meet the SQL fundamental requirements.
